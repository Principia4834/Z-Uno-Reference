begin()

initialise bus i2c/spi. Initialise sensor with given parameters / settings. Проверает id чипов. Подерживаемые id чипов:
  • 0x56 - BMP280
  • 0x57 - BMP280
  • 0x58 - BMP280
  • 0x60 - BME280
 По умолчанию выставяемые параметры setSampling. bpm.begin(addr, theWire, scl, sda)
 bpm: a variable of type BMP280.
 addr: адресс сенсора на шине i2c BME280_ADDRESS(0x77) и BME280_ADDRESS_ALTERNATE(0x76) - по умолчанию BME280_ADDRESS.
 theWire: указатель на обьект Wire - по умолчанию Wire.
 scl: pin SCL - по умолчанию SCL(9).
 sda: pin SDA - по умолчанию SDA(10).
 bool - true on success, false otherwise bpm.begin(spi, sck, miso, mosi, ss)
 bpm: a variable of type BMP280.
 theWire: указатель на обьект SPI .
 sck: custom SCK pin
 miso: custom MISO pin
 mosi: custom MOSI pin ss: custom SS pin bool - true on success, false otherwise
