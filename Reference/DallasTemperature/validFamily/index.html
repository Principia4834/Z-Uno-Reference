validFamily()

Checks the address on the matcher of the family of sensors the lib supports. sensors.validFamily(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: pointer to 8 byte address array.
 bool - returns true if address is of the family of sensors the lib supports.
