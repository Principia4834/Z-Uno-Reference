requestTemperaturesByIndex()

Sends command for one device to perform a temp conversion by index. На работу данной функции влияют: setCheckForConversion и setWaitForConversion sensors.requestTemperaturesByIndex(deviceIndex)
 sensors: a variable of type DallasTemperature.
 deviceIndex: индекс устройства.
 bool - true - otherwise, false - if device is disconnected
