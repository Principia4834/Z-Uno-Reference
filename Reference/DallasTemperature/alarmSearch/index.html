alarmSearch()

Search the wire for devices with active alarms. Данная функция актуальна при не выключенном REQUIRESALARMS sensors.alarmSearch(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив куда будет сохранен адресс.
 bool - true если успешно найденно устройствор, иначе false
