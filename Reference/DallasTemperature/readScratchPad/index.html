readScratchPad()

Считывает с ОЗУ данные датчика устройства. sensors.readScratchPad(deviceAddress, scratchPad)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс.
 scratchPad: указатель на структурру ScratchPad для считывания данных устройства.
 bool - returns true if данные были успешно считанны.
