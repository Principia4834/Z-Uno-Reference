requestTemperaturesByAddress()

Sends command for one device to perform a temperature conversion by address. На работу данной функции влияют: setCheckForConversion и setWaitForConversion sensors.requestTemperaturesByAddress(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс.
 bool - true - otherwise, false - if device is disconnected
