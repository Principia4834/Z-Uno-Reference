requestTemperaturesByAddress()

Sends command for one device to perform a temperature conversion by address. This function is affected by: setCheckForConversion and setWaitForConversion sensors.requestTemperaturesByAddress(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: pointer to an 8 byte array containing the address.
 bool - true - otherwise, false - if device is disconnected
