getHighAlarmTemp()

Returns current high alarm temperature for a device. Данная функция актуальна при не выключенном REQUIRESALARMS sensors.getHighAlarmTemp(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс или 0x0 - тогда отправляеться на все устройства. по умолчанию 0x0
 int8_t - valid in the range -55C - 125C or invalid DEVICE_DISCONNECTED
