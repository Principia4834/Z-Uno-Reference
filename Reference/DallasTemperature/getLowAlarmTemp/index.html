getLowAlarmTemp()

Returns current low alarm temperature for a device. Данная функция актуальна при не выключенном REQUIRESALARMS sensors.getLowAlarmTemp(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс
 int8_t - valid in the range -55C - 125C or invalid DEVICE_DISCONNECTED
