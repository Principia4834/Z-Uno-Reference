setWaitForConversion()

Устанавливает нужно ли при получение данных сенсора ждать покуда получишь или сразу выходить из функции. Другими словами включает асинхронный или синхронный режим. По умолчанию включен синхронный. sensors.setWaitForConversion(flag)
 sensors: a variable of type DallasTemperature.
 flag: true - включает синхронный режим работы, false включает ассинхронный режим.
 nothing
