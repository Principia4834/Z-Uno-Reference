setCheckForConversion()

Устанавливает режим проверки полученны ли данные от датчика или нет. sensors.setCheckForConversion(flag)
 sensors: a variable of type DallasTemperature.
 flag: true - в данном режиме постоянно провераеться готовность данных, false - ждет самое худшее время необходимое для полученния данных. по умолчанию true
 nothing
