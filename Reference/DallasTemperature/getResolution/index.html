getResolution()

Получает глобальное resolution. sensors.getResolution()
 sensors: a variable of type DallasTemperature.
 uint8_t - 9, 10, 11, or 12 bits Получает resolution для указанного адресса. sensors.getResolution(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс.
 uint8_t - 9, 10, 11, or 12 bits - 0 если ошибка.
