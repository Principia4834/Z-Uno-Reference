hasAlarm()

Проверает есть ли хоть одно устройство на шине поддерживаещее alarm. Данная функция актуальна при не выключенном REQUIRESALARMS sensors.hasAlarm()
 sensors: a variable of type DallasTemperature.
 bool - true - есть устройство, иначе false Проверает подерживает устройство с данным адрессом alarm или нет. Данная функция актуальна при не выключенном REQUIRESALARMS sensors.hasAlarm(deviceAddress)
 sensors: a variable of type DallasTemperature.
 deviceAddress: указатель на 8 байтный массив содержащий адресс.
 bool - true - устройство поддерживает, иначе false
