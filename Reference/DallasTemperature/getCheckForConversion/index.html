getCheckForConversion()

Получает текущий режим проверки полученны ли данные от датчика или нет. sensors.setCheckForConversion()
 sensors: a variable of type DallasTemperature.
 bool - true - в данном режиме постоянно провераеться готовность данных, false - ждет самое худшее время необходимое для полученния данных
