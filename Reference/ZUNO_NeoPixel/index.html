Zuno NeoPixel Library

Библиотека предназначена для работы с светодиодными лентами и неопиксельными кольцами, основанными на светодиодах WS2812 и WS2811. Биоблетка подерживает работу с несколькими сведиодными лентами. Количество подерживаемых лент зависит от обьема свободной RAM. Боллее подробней тут: addNeo Биоблетека может работать с любим пином, если выбранна подходящая перефирия. Боллее подробней тут: addNeo For more infomation about pins of Z-Uno here
  • addNeo
  • deleteNeo
 Note that NeoPixel uses one DMA channel. Note that NeoPixel использует разделяемую перефирию: USART0/USART1/USART2/TIMER0/TIMER1/WTIMER0 - по умолчанию используеться USART1. Одновременно может использоваться перефирия тоько одной биоблетекой. Custom pins can be used by calling addNeo()

#include "Arduino.h"
#include "ZUNO_NeoPixel.h"

/* LED counter */
#define LED_COUNT				150

/* LED pin number */
#define LED_PIN					9

/* the setup function runs once, when you press reset or power the board */
void setup() {
	NeoPixel.addNeo(LED_PIN, LED_COUNT);
}

/* the loop function runs over and over again forever */
void loop() {
	/* Turn on all LEDs */
	NeoPixel.fill(LED_PIN, 0, LED_COUNT, NeoPixel.RGB(255, 255, 0));
	/* Transferring colors to the ribbon */
	NeoPixel.show(LED_PIN); 
	delay(500); 
	/* Turn off all LEDs */
	NeoPixel.clear(LED_PIN);
	/* Transferring colors to the ribbon */
	NeoPixel.show(LED_PIN); 
	delay(500); 
}
