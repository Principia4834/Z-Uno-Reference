NeoPixel.gamma32()

A gamma-correction function for 32-bit packed RGB or WRGB colors. Makes color transitions appear more perceptially correct. NeoPixel.gamma32(x)
 x: RGB. 32 bit - Gamma-adjusted packed color, can then be passed in one of the setColor functions. Like gamma8, this uses a fixed gamma correction exponent of 2.6, which seems reasonably okay for average NeoPixels in average tasks. If you need finer control you'll need to provide your own gamma-correction function instead.
