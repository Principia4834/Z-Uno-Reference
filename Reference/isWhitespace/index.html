isWhitespace()

Analyse if a char is a space character. Returns true if the argument is a space or horizontal tab ('\t'). isWhitespace(thisChar)
 thisChar: variable. Allowed data types: char. true: if thisChar is a space character. 
		
if (isWhitespace(myChar)) { // tests if myChar is a space character
  Serial.println("The character is a space or tab");
}
else {
  Serial.println("The character is not a space or tab");
}
