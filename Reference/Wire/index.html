Wire Library

Implements I2C on Z-Uno pins 9–16. Default is 9 for SCL and 10 for SDA. Other pin range can be selected using ZUNO_REMAP_SPINS()
  • begin()
  • requestFrom()
  • beginTransmission()
  • endTransmission()
  • write()
  • available()
  • read()
  • enableTS()
 Z-Uno can only be I2C master. Default pins can be changed during I2C initialization, but they should be in fast mode): 
{literal}#define SCL_PIN 11
#define SDA_PIN 12

I2CDriver alternative_I2C(SCL_PIN, SDA_PIN); 

void setup() {
    // …
    Wire.bindDriver(&alternative_I2C); // This instructs to use other pins than default
    // …
}{/literal}
