Wire.begin()


Initiate the Wire library and join the I2C bus as a master. This should normally be called only once. Slave mode is not supported yet. 
Wire.begin();
None

Wire.begin()


Initiate the Wire library and join the I2C bus as a master or slave. This should normally be called only once. 

		Wire.begin()

		Wire.begin(address)

		Wire.begin(address, scl, sda) - код ошибки возращает.

		Wire.begin(address, scl, sda, *buffer, len) - код ошибки возращает.
address: the 7-bit slave address (optional); if not specified, join the bus as a master. Так же если адресс 0ч0 то тоже как мастер. scl: пин SCL. sda: пин SDA. buffer: указатель для буфера - в случае мастера это буфер для отправки данныйх slave; в случае slave половинна этого буффера используеться для отправки а другая для получения. len: размер буфера в байтах. nothing/код ошибки
