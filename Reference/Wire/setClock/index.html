Wire.setClock()

This function modifies the clock frequency for I2C communication. I2C slave devices have no minimum working clock frequency, however 100KHz is usually the baseline.. 
Wire.setClock(clockFrequency)
clockFrequency: the value (in Hertz) of desired communication clock. Accepted values are 10000 (low speed mode), 100000 (standard mode), 400000 (fast mode) and 1000000 (fast mode plus).
