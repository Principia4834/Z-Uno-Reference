Wire.setWireTimeout()

Данная функция устанавливает таймаут для I2C, по умолчанию значение 25 милисекунд. Данный таймаут позволяет "потупить" slave но в тоже время не дает master стопарнуться при проблеммах. При возниковении таймаута происходит сброс шины мастером. 
Wire.setWireTimeout(timeout)
timeout: значение таймаута в милисекундах.
