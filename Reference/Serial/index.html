Serial

Used for communication between the Z-Uno board and another microcontroller or a computer via UART or USB. Z-Uno boards have the following serial interfaces:
  • Serial — USB port
  • Serial0 — UART on TX0(24) and RX0(25) pins
  • Serial1 — UART on TX1(7) and RX1(8) pins
 Кроме пинов по умолчанию можно использовать и другие пины, устанавливаеться это с помощью метода begin()

Обратите внимание, что Serial использует буфер размером в 128 байта для приема, если его не достаточно то можете в begin() - указать свой. Обратите внимание, что Serial использует разделяемую перифирию, которую могут использовать и другие биоблитеке: SPI...
Когда библиотеки используют одну и туже перифирию - совместное использование их в таком случае не возможно. Используемая перифирия:
Serial ports Перифирия
Serial USART2
Serial0 USART0
Serial1 USART1
Обратите внимание, что Serial ports начинает постоянно использовать для приема один DMA канал от begin() до end()
Так же один DMA канал используеться во время передачи данных. All Serial ports have the following methods supported:
  • available()
  • begin()
  • end()
  • find()
  • findUntil()
  • flush()
  • parseFloat()
  • parseInt()
  • peek()
  • print()
  • printf()
  • println()
  • read()
  • readBytes()
  • readBytesUntil()
  • readString()
  • readStringUntil()
  • setTimeout()
  • write()
