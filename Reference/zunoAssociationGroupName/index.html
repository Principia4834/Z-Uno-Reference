zunoAssociationGroupName()

Callback function that returns name of desired group. By default names of groups are "User group 01", "User group 02" and etc... Z-Uno core calls this function when Z-Wave controller asks for group information by means of Association Group Information (AGI) command class. zunoAssociationGroupName(groupIndex)
 groupIndex: index of needed group. It starts from CTRL_GROUP_1... Pointer to nullterminated C-string. The maximum length of group name is 42 symbols! 
		
ZUNO_SETUP_ASSOCIATIONS(ZUNO_ASSOCIATION_GROUP_SET_VALUE, ZUNO_ASSOCIATION_GROUP_SET_VALUE_AND_DIM)

void setup() {
}

// returns custom names for association groups
const char *zunoAssociationGroupName(uint8_t groupIndex) {
	const char				*tmp;

	switch (groupIndex) {
		case CTRL_GROUP_1:
			tmp = "ZUNO_ASSOCIATION_GROUP_SET_VALUE";
			break;
		case CTRL_GROUP_2:
			tmp = "ZUNO_ASSOCIATION_GROUP_SET_VALUE_AND_DIM";
			break;
		default:
			tmp = NULL;
			break;
	}
	return (tmp);
}
