zunoAddChannel()

This function instructs Z-Uno to add one more channel during dynamic configuration. This function can be called many times to add more channels. Z-Uno supports up to 32 channels. You have to call this function between zunoStartDeviceConfiguration and zunoCommitCfg calls. zunoAddChannel(type, subtype, options) type
  • ZUNO_SWITCH_BINARY_CHANNEL_NUMBER
  • ZUNO_SWITCH_MULTILEVEL_CHANNEL_NUMBER
  • ZUNO_SENSOR_BINARY_CHANNEL_NUMBER
  • ZUNO_SENSOR_MULTILEVEL_CHANNEL_NUMBER
  • ZUNO_METER_CHANNEL_NUMBER
  • ZUNO_DOORLOCK_CHANNEL_NUMBER
  • ZUNO_FLOWSTOP_CHANNEL_NUMBER
  • ZUNO_SIREN_CHANNEL_NUMBER
  • ZUNO_BLINDS_CHANNEL_NUMBER
  • ZUNO_THERMOSTAT_CHANNEL_NUMBER
  • ZUNO_SWITCH_COLOR_CHANNEL_NUMBER
 subtype Sensor type. Applicable only for ZUNO_SENSOR_BINARY_CHANNEL_NUMBER, ZUNO_SENSOR_MULTILEVEL_CHANNEL_NUMBER and ZUNO_METER_CHANNEL_NUMBER. options One byte defining value size and value precision. Applicable only for ZUNO_SENSOR_BINARY_CHANNEL_NUMBER, ZUNO_SENSOR_MULTILEVEL_CHANNEL_NUMBER and ZUNO_METER_CHANNEL_NUMBER. Check ZWSupport directory for more details. Do not forget to enable all channel types you will use in your sketch with ZUNO_ENABLE(WITH_CC_...); Examples can be found in the description of zunoStartDeviceConfiguration.
