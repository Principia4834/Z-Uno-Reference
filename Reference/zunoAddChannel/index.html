zunoAddChannel()

This function instructs Z-Uno to add one more channel during dynamic configuration. This function can be called many times to add more channels. Максимальное поддержмваемое количество каналов - 32(от 0x0 до 31). Обычно каналы добавляют когда устройство не включенно. Чтобы успешно добавилось не забываем финилизировать с помощью zunoCommitCfg zunoAddChannel(type, subtype, options) type
  • ZUNO_SWITCH_BINARY_CHANNEL_NUMBER
  • ZUNO_SWITCH_MULTILEVEL_CHANNEL_NUMBER
  • ZUNO_SENSOR_BINARY_CHANNEL_NUMBER
  • ZUNO_SENSOR_MULTILEVEL_CHANNEL_NUMBER
  • ZUNO_METER_CHANNEL_NUMBER
  • ZUNO_DOORLOCK_CHANNEL_NUMBER
  • ZUNO_FLOWSTOP_CHANNEL_NUMBER
  • ZUNO_SIREN_CHANNEL_NUMBER
  • ZUNO_BLINDS_CHANNEL_NUMBER
  • ZUNO_THERMOSTAT_CHANNEL_NUMBER
  • ZUNO_SWITCH_COLOR_CHANNEL_NUMBER
 subtype Для большей информации смотрице описание каждого отдельного команд класса. options Для большей информации смотрице описание каждого отдельного команд класса Do not forget to enable all channel types you will use in your sketch with ZUNO_ENABLE(WITH_CC_...); Examples can be found in the description of zunoStartDeviceConfiguration.
