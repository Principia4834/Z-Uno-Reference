setBitOrder()

This function should not be used in new projects. Use SPISettings with beginTransaction() to configure SPI parameters.
Sets the order of the bits shifted out of and into the SPI bus, either LSBFIRST (least-significant bit first) or MSBFIRST (most-significant bit first). SPI.setBitOrder(order)
 order: either LSBFIRST or MSBFIRST.
