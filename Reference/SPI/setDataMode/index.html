setDataMode()

This function should not be used in new projects. Use SPISettings with beginTransaction() to configure SPI parameters.
Sets the SPI data mode: that is, clock polarity and phase. SPI.setDataMode(mode)
SPI.setDataMode(mode, slaveSelectPin)
 mode:
  • SPI_MODE0
  • SPI_MODE1
  • SPI_MODE2
  • SPI_MODE3
 slaveSelectPin: slave device SS pin.
