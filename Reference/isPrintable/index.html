isPrintable()

Analyse if a char is printable (that is any character that produces an output, even a blank space). Returns true if thisChar is printable. isPrintable(thisChar)
 thisChar: variable. Allowed data types: char. true: if thisChar is printable. 
		
if (isPrintable(myChar)) {  // tests if myChar is printable char
  Serial.println("The character is printable");
}
else {
  Serial.println("The character is not printable");
}
