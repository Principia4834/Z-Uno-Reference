zunoSetCustomWUPTimer()

Set custom time interval for the next wakeup. Function applicable to sleeping (EM2/EM4) and FLIRS devices. Look at ZUNO_SETUP_SLEEPING_MODE() for code example and details. 
zunoSetCustomWUPTimer(timeout)
 timeout: timeout in seconds. This specifies time interval before next wakeup. ZUNO_SETUP_SLEEPING_MODE()
