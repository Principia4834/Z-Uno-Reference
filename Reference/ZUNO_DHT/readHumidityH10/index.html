readHumidityH10()

Read Humidity in 10 th of percent. dht_sensor.readHumidityH10(bForce)
 dht_sensor: a variable of type DHT. bForce: false/true - false by default. If true then try to read the data even if the required time has not passed before the sensor can respond . int16_t - if error then 0x7FFF
