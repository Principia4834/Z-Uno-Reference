write()

Write a byte. If 'power' is one then the wire is held high at the end for parasitically powered devices. You are responsible for eventually depowering it by calling depower() or doing another read or write. Issue a 1-Wire rom skip command, to address all on bus. myWire.write(v, power)
 myWire: a variable of type OneWire. v: байт который передаем устройству. power: 1 или 0 - указывает оставить питание или нет по умолчанию 0 - не оставляет питание. nothing
