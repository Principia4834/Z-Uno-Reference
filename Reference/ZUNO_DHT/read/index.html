read()

Read value from sensor or return last one from less than two seconds. dht_sensor.read(bForce)
 dht_sensor: a variable of type DHT. bForce: false/true - по умолчанию false. Если true то пытаеться считать данные даже если не прошло нужное время прежде чем датчик сможет ответить. nothing/код ошибки
