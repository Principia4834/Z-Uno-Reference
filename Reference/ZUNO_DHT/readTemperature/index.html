readTemperature()

Read temperature. dht_sensor.readTemperature(S, bForce)
 dht_sensor: a variable of type DHT. S: false/true - по умолчанию false. Если true то температуру возратит в Fahrenheit иначе в Celcius. bForce: false/true - по умолчанию false. Если true то пытаеться считать данные даже если не прошло нужное время прежде чем датчик сможет ответить. float - Celcius/Fahrenheit - если ошибка то NaN
