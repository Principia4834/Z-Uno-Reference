malloc_stats()

Отображает информацию об используемой памяти malloc() Отображает с помощью Serial0, предварительно нужно иницилизировать с помощью begin() malloc_stats()
 nothing nothing 
		

#include <malloc.h>

void setup() {
	Serial0.begin(115200);//Иницилизируем Serial0 для вывода информации
}

void loop() {
	malloc_stats();
	Serial0.println();
	delay(1000);
}
