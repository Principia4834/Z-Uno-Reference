begin()

initialise bus i2c. Initialise sensor. Проверает id чипов. ccs.begin(addr, wire, scl, sda)
 ccs: a variable of type ZUNO_CCS811.
 addr: адресс сенсора на шине i2c CCS811_ADDRESS(0x5A) и CCS811_ADDRESS2(0x5B) - по умолчанию CCS811_ADDRESS.
 wire: указатель на обьект Wire - по умолчанию Wire.
 scl: pin SCL - по умолчанию SCL(9).
 sda: pin SDA - по умолчанию SDA(10).
 bool - true on success, false otherwise
