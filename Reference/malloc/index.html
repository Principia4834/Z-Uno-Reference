malloc()

The function allocates bytes and returns a pointer to the allocated memory. The memory is not initialized. If size is 0, then returns either NULL, or a unique pointer value that can later be successfully passed to free(). malloc(size)
 size: размер в байтах памяти которую нужно выделить. The functions return a pointer to the allocated memory, which is suitably aligned for any built-in type. On error, these functions return NULL. NULL may also be returned by a successful call to with a size of zero. Обратите внимание, RAM доступно 8192 байта, malloc использует данную память на равне со скетчем, но по остаточному принципу. Например скетч использует: 805 байта RAM, тогда malloc останеться 7387 байта. malloc использует для своих нужд при выдилении 4 байта, а так же выравнивает на 4 байта любое выдиление. Например если есть жилание выделить 1 байт памяти, то по факту malloc использует 8 байт RAM
