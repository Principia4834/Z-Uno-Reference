malloc()

The function allocates bytes and returns a pointer to the allocated memory. The memory is not initialized. If size is 0, then returns either NULL, or a unique pointer value that can later be successfully passed to free(). malloc(size)
 size: размер в байтах памяти которую нужно выделить. The functions return a pointer to the allocated memory, which is suitably aligned for any built-in type. On error, these functions return NULL. NULL may also be returned by a successful call to with a size of zero.
