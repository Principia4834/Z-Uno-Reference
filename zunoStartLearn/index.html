zunoStartLearn()

Enable and abort Learn mode. Learn mode is used to include Z-Uno in your Z-Wave network and exclude it from existing network. This function can be used to include or exclude Z-Uno instead of triple click on the Service Button. zunoStartLearn(timeout, secure) timeout time in seconds to wait for inclusion/exclusion before canceling. secure 0 to start unsecure inclusion (skipping all security and encryption) and > 0 for secure inclision (with Security S0 or S2 depending on the settings in the IDE.
