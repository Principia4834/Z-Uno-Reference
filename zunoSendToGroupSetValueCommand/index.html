zunoSendToGroupSetValueCommand()

This function sends switch on/off commands (using Basic Command Class) to devices in association group defined as ZUNO_ASSOCIATION_GROUP_SET_VALUE or ZUNO_ASSOCIATION_GROUP_SET_VALUE_AND_DIM (see ZUNO_SETUP_ASSOCIATIONS). zunoSendToGroupSetValueCommand(CTRL_GROUP_n, value) CTRL_GROUP_n number of group: CTRL_GROUP_1, ... CTRL_GROUP_5 value 0 (off) or 255 (on) 
        zunoSendToGroupSetValueCommand(CTRL_GROUP_1, 50); // Dim to 50%
