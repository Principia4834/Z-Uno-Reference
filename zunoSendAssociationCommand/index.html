zunoSendAssociationCommand()

This function instructs Z-Uno to send a control sequence to an association group. This is a more generic function than zunoSendToGroupSetValueCommand(), zunoSendToGroupDimmingCommand(), zunoSendToGroupScene(), zunoSendToGroupDoorlockControl(). Association groups are defined via macro ZUNO_SETUP_ASSOCIATIONS() or via dynamic function zunoAddAssociation(). zunoSendAssociationCommand(CTRL_GROUP_n, group_type, value1, value2) CTRL_GROUP_n number of group: CTRL_GROUP_1, ... CTRL_GROUP_5 group_type Type of the command to send: ZUNO_ASSOC_BASIC_SET_NUMBER, ZUNO_ASSOC_BASIC_SET_AND_DIM_NUMBER, ZUNO_ASSOC_SCENE_ACTIVATION_NUMBER, ZUNO_ASSOC_DOORLOCK_CONTROL_NUMBER as defined in ZUNO_Definitions.h value1 Dimming value, scene Id or direction value2 start or stop for dimming (otherwise ignored) It is recommended to use one of zunoSendToGroupSetValueCommand(), zunoSendToGroupDimmingCommand(), zunoSendToGroupScene(), zunoSendToGroupDoorlockControl() instead.
