Radiation, Temperature and Humidity Monitor with LCD Display

Andreas Drollinger (MIT License) The project connects together the following hardware components:
  • Z-Uno
  • Radiation monitor — RM-60/70/80 radiation monitor from AWARE Electronics (UART)
  • Temperature/humidity sensor — DHT22 temp/hum sensor (one pin connection)
  • LCD display — Adafruit Sharp 1.3" Memory LCD Display (SPI)
  • Control buttons — two simple press buttons connected to two digital IOs
Radiation data, temperature and humidity can be reported every minutes via Z-Wave. Radiation events can be indicated via the Z-Uno user LED. Status information can be reported via the USB attached serial interface

Configuration options: Z-Wave reporting, LED activity and status reporting via serial interface can individually be enabled or disabled. Configurations are stored in the EEPROM and restored after power cycling.
