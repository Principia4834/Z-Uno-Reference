single line comment

Comments are lines in the program that are used to inform yourself or others about the way the program works. They are ignored by the compiler, and not exported to the processor Comments only purpose are to help you understand (or remember) how your program works or to inform others how your program works. There are two different ways of marking a line as a comment: single line comment and multi-line comments 
x = 5;  // This is a single line comment. Anything after the slashes is a comment 
        // to the end of the line
