10 Channel certified example

This page contains specific settings option for certification sketch. All standard Z-Uno parameters and Z-Wave descriptions are listed on this page. It consist of a very complex mains power device with 3 relay switches, 3 LED dimmers, temperature, humidity and motion sensors as well as reed switch.

Associations

Device has 2 association groups. See the table below.
# Name Description Maximum number of nodes Commands
1 Lifeline Main group for device reports. See Z-Wave specification for more details. 5 DeviceResetLocally.Report, SwitchBinary.Report, SwitchMultilevel.Report, SensorMultilevel.Report, Notification.Report
2 User group 02 Here motion sensor sends Basic on/off commands every time it is triggered. 5 Basic.On, Basic.Off

Configuration parameters

In addition to the Z-Uno system configuration parameters. This sketch provides user-side configuration parameters. See the table below.
# Description Size Default value Possible values
64 Temperature hysteresis for sensor reports. This parameter is defined in 10th of degree. 4 bytes 5 1–20
65 Humitity hysteresis for sensor reports. This parameter is defined in 10th of percent. 4 bytes 5 1–20
66 Motion sensor's retrigger interval in milliseconds 4 bytes 5000 0–100000

S2 Levels

This sketch supports these S2 keys:
  • S0
  • S2_UNAUTHENTICATED
  • S2_AUTHENTICATED
