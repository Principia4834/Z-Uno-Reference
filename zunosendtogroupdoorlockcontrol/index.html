zunoSendToGroupDoorlockControl()

This function sends open/close commands (using Door Lock Command Class) to devices in association group defined as ZUNO_ASSOCIATION_GROUP_DOORLOCK (see ZUNO_SETUP_ASSOCIATIONS). zunoSendToGroupDoorlockControl(CTRL_GROUP_n, value) CTRL_GROUP_n number of group: CTRL_GROUP_1, ... CTRL_GROUP_5 value 0 (close) or 255 (open) 
zunoSendToGroupDoorlockControl(CTRL_GROUP_1, 255); // Open
