pow(), exp(), log(), log10()

Calculates the value of a number raised to a power or logarithm of a value. pow() can be used to raise a number to a fractional power. This is useful for generating exponential mapping of values or curves. 
value = pow(base, exponent);
value = exp(exponent);
exponent = log(value);
exponent = log10(value);
base the number (float) exponent the power to which the base is raised (float) value the base rised in power of exponent (float) The result of the exponentiation or logarithm (float)
